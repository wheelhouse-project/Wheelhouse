Control your Windows PC with your voice — completely hands-free.

WheelHouse is free, open-source voice control built for people who need it. If using a keyboard and mouse is painful, difficult, or impossible, WheelHouse aims to give you the whole computer by voice: speak to type into any program, run commands, and click buttons by saying their names. No account, no subscription, and with the default settings your voice never leaves your machine.

What you can do

  • Dictate anywhere

    Speak into whatever program has focus — email, documents, chat, web forms — and the words stream in as you talk, usually starting in under two seconds. Say comma or new line and WheelHouse types the punctuation for you.

  • Command your computer

    x-ray activate chrome, x-ray notepad, select all — WheelHouse ships with a catalog of commands, and a built-in Pattern Manager lets you add your own phrases without touching any code. Commands that launch or switch programs start with the spoken word x-ray, so everyday dictation never triggers them by accident.

  • Click things by name

    Say click cancel or click the submit button and WheelHouse finds that control in the focused window and clicks it. If a control has no useful name, say apply numbers to put a number on everything clickable, then click 5.

  • Private by default

    The default speech engine runs entirely on your machine, on the CPU — no cloud account, no audio leaving your computer, and no telemetry of any kind. Dictated text is even kept out of WheelHouse's own log files.

  • Careful about where text goes

    Before typing a single word, WheelHouse checks that the thing you are focused on actually accepts text — so dictation does not spill keystrokes into the wrong window.

Install WheelHouse

You need: Windows 10 or 11 (64-bit), a microphone, and a few gigabytes of free disk space for the speech model.

  1. Open PowerShell. Press the Start button, type powershell, and press Enter. A window with a text prompt opens.
  2. Paste this command and press Enter: 
    irm https://github.com/wheelhouse-project/WheelHouse/releases/latest/download/install-wheelhouse.ps1 | iex
    The installer checks your hardware, sets up its own private Python environment (it installs nothing system-wide), downloads the offline speech model, and adds WheelHouse to your Start menu.
  3. Start WheelHouse from the Start menu and start talking.

About security warnings: WheelHouse is currently unsigned — the project does not yet have a code-signing certificate — so Windows SmartScreen or your antivirus may warn you the first time. The warning is about the missing signature, not about what the software does: the complete source code is public, and the install guide explains each warning and how to proceed.

Full details — optional speech engines, troubleshooting, and how to uninstall — are in the install guide.

Choose where speech is processed

EngineWhere your speech goesWhen to choose it
Parakeet (default) Stays on your machine (CPU) No account, no cloud, works offline. The default for everyone.
Distil-Whisper (opt-in) Stays on your machine (NVIDIA GPU) You have a CUDA-capable graphics card and want lower latency.
Google Cloud (opt-in) Streams to Google's servers You have a Google Cloud account and prefer its recognition quality.

Questions people ask

Does it need an internet connection?

Only for the initial download. After installing, the default speech engine works completely offline. Internet is needed again only if you opt into the Google Cloud engine or the optional AI features.

Is it really free?

Yes. WheelHouse is open source under the Apache 2.0 license. There is no account, no subscription, no paid tier, and no advertising.

Is my voice recorded or sent anywhere?

With the default engine, no — audio and recognized text never leave your machine, and WheelHouse reports nothing to the project or anyone else. The engines and features that do make network connections are opt-in, and the privacy document states exactly what each one sends.

How do I uninstall it?

The installer doubles as the uninstaller and can keep or remove your personal settings — see the install guide for the exact steps.

Something didn't work. Where do I get help?

Please open an issue on the GitHub issue tracker — reports from people who depend on hands-free input are especially valuable. WheelHouse is a young project from a single primary author: it has been the author's daily driver for years, but it has met only a small set of machines so far, so expect some rough edges on hardware it has not seen yet.
